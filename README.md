GDINA Package for Cognitive Diagnosis Modelling with Polytomous Attributes

Features of the package
-----------------------
-   Estimating PGDINA model and a variety of widely-used models subsumed by the PGDINA model for polytomous attributes, including the pDINA model, pDINO model, and pA-CDM
-   Validating Q-matrix with polytomous attributes under the general model framework
